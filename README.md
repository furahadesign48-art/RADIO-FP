# Fréquence Positive - Application Radio

Application mobile officielle de la radio **Fréquence Positive**, animée par **Le Neveu National**.

## Fonctionnalités
- 📻 Écoute en direct haute qualité.
- 🎵 Affichage dynamique des titres et artistes en temps réel (via Radio King API).
- 📱 Support du mode arrière-plan et des contrôles sur l'écran de verrouillage.
- 🌓 Modes Sombre et Clair avec plusieurs thèmes colorés.
- ⏲️ Minuterie de mise en veille (Sleep Timer).
- 🔗 Liens vers les réseaux sociaux (Facebook, Instagram).

## Installation
1. Clonez le dépôt : `git clone https://github.com/votre-pseudo/RADIO-FP.git`
2. Installez les dépendances : `npm install`
3. Lancez l'application : `npx expo start`
